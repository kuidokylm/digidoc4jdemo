DigiDoc4j hwcrypto demo webapp

This is a Demo webapp for digital signing in TEST environment with the combination of hwcrypto.js on the frontend and DigiDoc4j on the backend.

Preparations:

  1. Install your country specific eID driver software.
  2. Install Chrome Token Signing Software for giving eSignatures in Google Chrome.
  3. Make sure you are accessing the page over HTTPS.
  4. Make sure you are using test ID-card.

Peek into JavaScript console for more verbose background information.
